B. Sektore Ozel Zorluklar

Havacilik dijital donusumunde karsilasilan teknik ve regulatorik zorluklar

GDS Bagimliligi vs NDC Gecisi

Geleneksel GDS (Amadeus, Sabre, Travelport) araciligi modeli yuksek komisyon maliyeti demek. IATA'nin NDC (New Distribution Capability) standardi direkt dagitim imkani sunuyor. Ancak gecis sureci karmasik: mevcut GDS baglantilari korunurken NDC katmani eklenmeli. SunExpress'in Navitaire PSS'i NDC uyumlu ama tam entegrasyon henuz yapilmamis.

Real-Time Envanter Yonetimi

Ucus koltuk envanteri saniye bazinda degisiyor. PSS ile mobil uygulama arasindaki senkronizasyon gecikme (latency) kritik. Overselling riski vs revenue maximization dengesi. Cache stratejisi + eventual consistency modeli gerekiyor.

PCI-DSS Odeme Guvenligi

Havayolu odeme islemleri PCI-DSS Level 1 uyumluluk gerektiriyor. Kredi karti verisi islemede 3D Secure 2.0 zorunlu. Tokenization, end-to-end encryption, fraud detection. Apple Pay/Google Pay entegrasyonu PCI yukunu azaltiyor — tercih edilmeli.

GDPR + KVKK Uyumlulugu

SunExpress hem Turkiye (KVKK) hem Avrupa (GDPR) hukuku altinda. Yolcu verisi isleme, consent yonetimi, veri silme hakki, cross-border veri transferi. Almanya'daki operasyonlar ek regulatorik yuk getiriyor. Privacy-by-design mimari sart.

Disruption Yonetimi

Gecikme, iptal, yonlendirme (diversion) — havacilik sektorunun en buyuk musteri memnuniyetsizlik kaynagi. EU 261/2004 yolcu haklari regulasyonu. Proaktif bilgilendirme, otomatik rebooking, tazminat surec yonetimi. Real-time veri isleme ve karar motoru gerekiyor.

Cok Dilli / Cok Para Birimli Destek

SunExpress minimum 3 dil (TR, EN, DE) desteklemeli. Almanya pazari icin EUR, Turkiye icin TRY, digerleri icin USD/GBP. Doviz kuru guncellemesi, farkli vergi rejimleri (KDV vs VAT), lokalize icerik yonetimi. i18n/l10n mimarisi bastan dogru kurulmali.

Sezonalite Yonetimi

SunExpress turizm odakli — Haziran-Eylul doneminde trafik 4-5x artiyor. Altyapi olcekleme (auto-scaling), yuk testi, kapasite planlama. Yaz sezonu oncesi stress test zorunlu. CDN ve edge computing ile global dagitim.

Siber Guvenlik

Havacilik sektoru siber saldirilarin yuksek hedefi. 2024'te global havayollarinda %35 saldiri artisi. DDoS, credential stuffing, API abuse, veri sizintisi riskleri. Zero-trust mimari, API rate limiting, WAF, SIEM/SOC entegrasyonu.

D. Teknoloji Trendleri

Havacilik dijital donusumunu sekillendiren teknoloji trendleri

NDC (New Distribution Capability) Donusumu

IATA'nin NDC standardi, havayollarinin GDS aracisi olmadan zengin icerik (gorsel, paket, kisisel teklif) dagitmasini sagliyor. 2025 itibariyle NDC islem hacmi %40+ buyudu. SunExpress'in Navitaire altyapisi NDC uyumlu — tam entegrasyon ile OTA kanallarinda daha yuksek donusum orani ve dusuk dagitim maliyeti.

Yuksek Oncelik

AI-Powered Dinamik Fiyatlama

Makine ogrenmesi ile talep tahmini, rakip fiyat izleme, kisisel fiyat optimizasyonu. Ryanair bu alanda sektor lideri — AI ile koltuk basi geliri %8 artmis. SunExpress icin potansiyel: yolcu segmentine gore dinamik paket fiyatlamasi, ancillary upsell zamanlama optimizasyonu.

Orta Vade

Predictive Disruption Management

Hava durumu, hava trafigi, ekip planlama verilerini birlestirerek gecikme/iptal tahmini. Yolcuya ucus oncesi proaktif bilgilendirme, otomatik alternatif onerisi. easyJet bu alanda yatirim yapiyor — gecikme bildirimini 30 dk onceden veriyor.

Yuksek Oncelik

Biyometrik Pasaport Kontrolu

Yuz tanima ile havaalani check-in kiosk, guvenlik gecisi ve binis kapisi. SITA Smart Path altyapisi 40+ havalimaninda aktif. Istanbul Havalimani pilot programi planlaniyor. Mobil uygulamada biyometrik on-kayit — havalimaninda cuzdansiz seyahat.

Uzun Vade

Surdurulebilirlik Dashboard

Yolcu bazinda karbon ayak izi hesabi, ofset secenegi, surdurulebilir yakit (SAF) bilgisi. EU regulasyonlari ile zorunlu hale geliyor. Lufthansa Grubu 2030 hedefleri SunExpress'i de kapsiyor. Mobil uygulamada seffaf karbon izleme — marka degeri artisi.

Orta Vade

Super App Donusumu

Ucus + otel + arac + sigorta + transfer + deneyimler — tek uygulamada uctan uca seyahat planlamasi. Grab (Guneydogu Asya), WeChat (Cin) modelinden esinlenen havacilik super app'leri. Ryanair ve easyJet bu yonde ilerliyor. SunExpress icin turizm odakli super app potansiyeli yuksek.

E. Loodos Deger Onerisi

SunExpress dijital donusum projesinde Loodos'un sunacagi teknik ve stratejik deger

Native iOS (SwiftUI) + Android (Kotlin Compose)

  • SwiftUI + MVVM + @Observable ile modern iOS mimarisi
  • Kotlin Compose + MVI ile modern Android mimarisi
  • iOS 17+ / Android 13+ hedefleme — en yeni API'lar
  • %40-50 performans artisi (React Native'e gore)
  • Platform-native animasyon, haptic feedback, widget destegi
  • Live Activity, Dynamic Island, App Intents entegrasyonu

GDS/PSS API Entegrasyon Deneyimi

  • Navitaire New Skies SOAP/XML API entegrasyon bilgisi
  • Amadeus Altea REST API deneyimi
  • IATA NDC Level 3-4 implementasyon
  • PSS-to-REST middleware katmani tasarimi
  • Real-time envanter senkronizasyonu
  • GDS fallback + circuit breaker pattern'lari

Real-Time Veri Mimarisi

  • WebSocket + Server-Sent Events ile canli ucus takibi
  • Redis + Event sourcing ile envanter cache
  • Apache Kafka / AWS Kinesis ile event streaming
  • Push notification altyapisi (FCM + APNs)
  • Disruption detection + otomatik rebooking motoru
  • Edge computing ile dusuk latency garanti

Apple Wallet / Google Wallet Entegrasyonu

  • PKPass / Google Pass JSON olusturma ve dagitim
  • Dinamik pass guncelleme (gate, gecikme, iptal)
  • Push notification ile pass update trigger
  • Markasal pass tasarimi (logo, renk, layout)
  • QR + NFC ikili tarama destegi
  • Boarding pass ekran parlaklik otimizasyonu

AI-Powered Oneri Motorlari

  • Kisisel ucus ve fiyat onerisi (collaborative filtering)
  • Kontekstel ancillary upsell (rota, sezon, segment bazli)
  • Dogal dil seyahat asistani (LLM tabanli)
  • Predictive disruption management
  • A/B testing + bayesian optimizasyon altyapisi
  • ML model serving (on-device + server-side)

Performance-First Mimari

  • Offline-first: boarding pass, rezervasyon bilgisi, ucus detaylari
  • Edge caching: CDN + service worker + local cache
  • Optimistic UI: aninda kullanici geri bildirimi
  • Code splitting + lazy loading + pre-fetching
  • Image optimization: WebP + progressive loading
  • Cold start optimizasyonu: <2 saniye hedefi

KVKK ve GDPR Uyumlu Veri Yonetimi

Veri Koruma Mimarisi

  • Privacy-by-design: veri minimizasyonu, amac sinirlamasi
  • Consent Management Platform (CMP) entegrasyonu
  • Data residency: TR verisi TR'de, EU verisi EU'da
  • Right to erasure (unutulma hakki) otomasyonu
  • Data Processing Agreement (DPA) sablon yonetimi
  • Veri erisim loglama ve audit trail

Guvenlik Katmanlari

  • End-to-end encryption (TLS 1.3 + AES-256)
  • API security: OAuth 2.0 + JWT + rate limiting
  • Certificate pinning (mobil uygulamada)
  • PCI-DSS Level 1 odeme tokenization
  • WAF + DDoS korumasi (Cloudflare / Akamai)
  • Penetration testing + bug bounty programi

Neden Simdi?

Pazar Dinamikleri

  • Pegasus fark aciyor: Her ceyrek yeni dijital ozellik ekliyor. Gecikme = pazar kaybi.
  • Yolcu beklentisi yukseliyor: Ryanair/easyJet deneyimi Avrupa yolcusunu sartlandirdi.
  • OTA baskisi artiyor: Booking, Google Flights — direkt kanal guclenmeli.
  • Yaz sezonu 2027: Simdi baslarsa MVP yaz sezonuna yetisir.

Teknik Zorunluluklar

  • React Native teknoloji borcu: Guncellemeler zorlasti, performans sinirlari geldi.
  • iOS 18+ API'lari: Live Activity, App Intents, WidgetKit — sadece native'de.
  • Guvenlik gereksinimleri: PCI-DSS 4.0 gecis tarihi yaklasti.
  • NDC hazirlik: Navitaire NDC katmani hazir — uygulama tarafini tamamlayin.

Bir Sonraki Adim

SunExpress dijital donusum projesini birlikte sekillendirelim. Teknik PoC (Proof of Concept) ile baslayabiliriz.
