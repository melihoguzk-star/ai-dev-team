YÜKSEK ikas API Bağımlılığı

Tüm backend ikas'ın GraphQL API'sine bağlı. API değişiklikleri doğrudan mobil uygulamayı etkiler. Rate limit veya breaking change riski.

YÜKSEK API Key Yönetimi

GraphQL API key client-side HTML'de mevcuttu. iOS uygulamasında bu key Keychain'de saklanmalı, obfuscation uygulanmalı.

ORTA Captcha Uyumu

Web'de reCAPTCHA kullanılıyor. Mobile'da reCAPTCHA v3 veya hCaptcha iOS SDK'sı entegre edilmeli; aksi hâlde auth mutasyonları çalışmaz.

ORTA Ödeme Gateway Çeşitliliği

6+ farklı ödeme yöntemi. Her birinin iOS SDK entegrasyonu ayrı iş yükü. Başlangıç için İyzico + Kredi Kartı yeterli olabilir.

ORTA Sosyal Login Apple Gerekliliği

App Store kuralı: eğer sosyal login sunuluyorsa Sign in with Apple zorunludur.

ORTA GraphQL Schema Değişimi

ikas SaaS olduğundan schema bildirimsiz değişebilir. Hata toleransı yüksek veri modeli tasarlanmalı.

DÜŞÜK Yotpo Widget Entegrasyonu

Yotpo'nun native iOS SDK'sı veya API'si ayrıca incelenmeli; web widget iOS'ta doğrudan çalışmaz.

DÜŞÜK Kampanya Karmaşıklığı

BuyXGetY, CampaignOffer, Raffle, LoyaltyPoints aynı anda aktif olabilir. Sepet hesaplaması sunucu tarafında yapılıyor (bu avantaj).

DÜŞÜK KVKK / Veri Gizliliği

Kullanıcı kişisel verisi işleniyor. iOS App Store Privacy Nutrition Label doğru doldurulmalı.

BİLGİ Erişilemeyen Sayfalar

/cart, /checkout, /account ve /account/* sayfaları robots.txt ile engellenmiş; statik analiz yapılamadı. İçerikleri tahmin bazlıdır.

Bu rapor pasif web analizi yöntemleriyle hazırlanmıştır (HTML kaynak kodu, sitemap, robots.txt, Schema.org, GraphQL introspection).
Herhangi bir kimlik bilgisi loglanmamış, siteye zarar verecek istek yapılmamıştır.
"Tahmin" olarak işaretlenen bilgiler doğrudan gözlemlenememiş, mantıksal çıkarımla elde edilmiştir.
